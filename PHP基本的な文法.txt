変数、データ型、演算子など


 1. 変数

変数はデータを格納するための場所です。PHPでは、変数名の前にドル記号（$）を付けます。

php

<?php
$myVariable = "Hello, World!";
$number = 42;
?>


 ルール
 変数名はドル記号（$）で始める。
 変数名は英文字、数字、アンダースコア（_）を含むことができる。
数字で始めることはできない。
 変数名は大文字と小文字を区別する。
【$Variable と $variable は異なる変数。】

 2. データ型

PHPは動的型付け言語であり、変数に格納されるデータの型を明示的に宣言する必要はない。

主なデータ型

*文字列 (string): 文字の集合

*整数 (integer: 整数値

*浮動小数点数 (float): 小数点数

*ブール値 (boolean): 真 (true) または 偽 (false)

*配列 (array): 複数の値のコレクション

*オブジェクト (object): クラスのインスタンス

*NULL: 特別な値、変数に値が設定されていないことを表す

 例

php
<?php
$string = "Hello, PHP!";
$integer = 100;
$float = 10.5;
$boolean = true;
$array = array("apple", "banana", "cherry");
$object = (object) ['property' => 'value'];
$nullValue = NULL;
?>


 3. 演算子

PHPには多くの演算子があります。以下に主な演算子を示します。

 算術演算子
 (加算): $a + $b
 (減算): $a - $b
 (乗算): $a * $b
 (除算): $a / $b
 (剰余): $a % $b

php
<?php
$a = 10;
$b = 3;
echo $a + $b; // 13
echo $a - $b; // 7
echo $a * $b; // 30
echo $a / $b; // 3.3333...
echo $a % $b; // 1
?>


 代入演算子
 (代入): $a = $b
 (加算代入): $a += $b
 (減算代入): $a -= $b
 (乗算代入): $a \*= $b
 (除算代入): $a /= $b
 (剰余代入): $a %= $b

php
<?php
$a = 5;
$b = 2;
$a += $b; // $a は 7 になる
$a -= $b; // $a は 5 になる
$a *= $b; // $a は 10 になる
$a /= $b; // $a は 5 になる
$a %= $b; // $a は 1 になる
?>


 比較演算子
 (等しい): $a == $b
 (等しくない): $a != $b
 (小さい): $a < $b
 (大きい): $a > $b
 (以下): $a <= $b
 (以上): $a >= $b

php
<?php
$a = 5;
$b = 10;
var_dump($a == $b); // bool(false)
var_dump($a != $b); // bool(true)
var_dump($a < $b);  // bool(true)
var_dump($a > $b);  // bool(false)
var_dump($a <= $b); // bool(true)
var_dump($a >= $b); // bool(false)
?>


 論理演算子
 (AND): $a && $b
 (OR): $a || $b
 (NOT): !$a

php
<?php
$a = true;
$b = false;
var_dump($a && $b); // bool(false)
var_dump($a || $b); // bool(true)
var_dump(!$a);      // bool(false)
?>
